Bootstrap

Sleek, intuitive, and powerful mobile first front-end framework for faster and easier web development.

http://getbootstrap.com/

What's Bootstrap

  • A free collection of tools for creating websites and web applications
  • CSS
  • Javascript

History

  • Developed by Mark Otto and Jacob Thornton at Twitter
  • In August, 2011 Twitter released Bootstrap as open source
  • In February 2012, it was the most popular GitHub development project
  • Used by ASP.Net , Joomla3

Why Bootstrap

  • Compatible to most of the browser
  • Nice Design
  • Responsive
  • Quick front end development

Download

  • LESS: http://getbootstrap.com/
  • SASS: https://github.com/jlong/sass-bootstrap

Simply import the css and js files into html
